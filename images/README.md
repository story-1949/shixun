# 效果图目录

本目录用于存放项目的效果图截图。

## 📸 需要的截图

请按照以下顺序截图并保存到此目录：

1. **main-interface.png** - 主界面截图
   - 启动程序后的初始界面
   - 展示所有功能区域

2. **factory-pattern.png** - 工厂模式演示
   - 点击"添加计算机/服务器/打印机"后的效果
   - 显示设备创建的详细信息

3. **device-list.png** - 设备列表展示
   - 点击"显示所有设备"后的效果
   - 展示表格化的设备列表

4. **decorator-pattern.png** - 装饰器模式演示
   - 点击"装饰设备"后的效果
   - 显示原始设备和装饰后的对比

5. **strategy-pattern.png** - 策略模式演示
   - 点击"日常维护"或"深度维护"后的效果
   - 展示不同策略的执行结果

6. **observer-pattern.png** - 观察者模式演示
   - 点击"更新状态"后的效果
   - 显示状态选择对话框和通知结果

7. **console-demo.png** - 控制台演示
   - 运行 TestDemo 的控制台输出
   - 展示五种设计模式的完整演示

## 📝 截图建议

- 使用 Windows 截图工具（Win + Shift + S）
- 确保截图清晰，分辨率适中
- 截图时包含完整的窗口内容
- 建议使用 PNG 格式保存

## 🔄 如何添加截图

1. 运行程序：`java -cp bin com.device.Main`
2. 按照上述列表进行操作并截图
3. 将截图保存到此目录，使用对应的文件名
4. 提交到 Git 仓库

## ⚠️ 注意事项

如果暂时没有截图，README 中已经添加了占位符和详细说明，用户仍然可以通过文字描述了解系统功能。
